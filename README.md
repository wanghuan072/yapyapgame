# YAPYAP Game

**YAPYAP** is a multiplayer horror game on Steam that combines voice-activated spellcasting with co-op chaos and stealth escape mechanics.

## About the Game

YAPYAP is a unique multiplayer horror experience where players act as minions infiltrating a rival tower. The game features:

- **Voice-Activated Spells**: Speak spell commands into your microphone to trigger magical effects
- **Co-op Gameplay**: Team up with up to 5 players in online multiplayer
- **Puzzle Solving**: Navigate through challenging puzzles including ball puzzles, balance puzzles, and door puzzles
- **Stealth & Survival**: Escape monsters while smashing high-value objects and avoiding detection
- **Vandalism Mechanics**: Destroy objects and cause chaos in the tower
